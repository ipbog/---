# Отслеживание Багов

Этот файл используется для отслеживания известных багов в проекте Model-Hub.

## Формат Записи о Баге

*   **ID Бага:** (Уникальный идентификатор, например, BUG-001)
*   **Дата Обнаружения:** YYYY-MM-DD
*   **Версия Проекта:** (Версия, в которой баг был обнаружен)
*   **Компонент(ы):** (Например, `core_burn`, `gui_app`, `lsp_server_backend`)
*   **Описание Бага:** (Подробное описание проблемы)
*   **Шаги для Воспроизведения:**
    1.  ...
    2.  ...
*   **Ожидаемое Поведение:**
*   **Фактическое Поведение:**
*   **Приоритет:** (Низкий, Средний, Высокий, Критический)
*   **Статус:** (Открыт, В Работе, Исправлен, Закрыт, Отложен)
*   **Ответственный:** (Кто работает над багом)
*   **Комментарии/Заметки:** (Дополнительная информация)
