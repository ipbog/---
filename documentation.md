# Документация по Проекту Model-Hub

Этот документ содержит техническую документацию по проекту AI Code Assistant "Model-Hub".

## 1. Архитектура Системы

*   Обзор общей архитектуры.
*   Взаимодействие между крейтами:
    *   `utils_crate`
    *   `core_burn`
    *   `model_loader`
    *   `inference_engine`
    *   `lsp_server_backend`
    *   `gui_app`
    *   `cli_app`
*   Поток данных при выполнении запроса (например, автодополнения).

## 2. Описание Крейтов

### 2.1. `utils_crate`
    *   Назначение.
    *   Основные структуры (`InferenceTask`, `SamplerParams`).

### 2.2. `core_burn`
    *   Назначение.
    *   Архитектура `GemmaModel`.
    *   Реализация `KvCache` и `RoPE`.
    *   API для `forward_with_cache`.

### 2.3. `model_loader`
    *   Назначение.
    *   Процесс загрузки моделей (SafeTensors, MPK).
    *   Работа с `GemmaModelConfig`.

### 2.4. `inference_engine`
    *   Назначение.
    *   Структура `InferenceRunner` и `ModelContext`.
    *   Работа с `InferenceTask` и `SamplerParams`.
    *   Алгоритм сэмплинга в `SimpleSampler`.

### 2.5. `lsp_server_backend`
    *   Назначение.
    *   Реализация LSP протокола.
    *   Обработка ключевых запросов (initialize, completion, hover, didOpen/Change/Close).
    *   Взаимодействие с `inference_engine`.

### 2.6. `gui_app`
    *   Назначение.
    *   Архитектура на `egui`/`eframe`.
    *   Взаимодействие с `lsp_server_backend` через `LspClient`.
    *   Основные компоненты UI.

### 2.7. `cli_app`
    *   Назначение.
    *   Основные команды и опции.
